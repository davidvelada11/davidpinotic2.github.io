INTRODUCCIÓN A CSS

David Pino

Una vez que tenemos el contenido deseado en nuestra página web, el siguiente paso es darle un toque personal. Para ello, necesitaremos cambiar el lenguaje HTML por el de CSS.
Lo podemos personalizar de distintas maneras:

Cambiando el estilo de los elementos de una etiqueta

Ejemplo: Todos los párrafos de esta página tendrán el rojo como color de fuente y la página tendrá un fondo azul con un tono oscuro.

Cambiando el estilo de un elemento concreto

En este caso, voy a hacer que solo el encabezado de este apartado cambie su color de fuente a un tono verde pistacho.

Cambiando el estilo de varios elementos concretos

Ahora, voy a hacer que solo el fondo de los párrafos de los encabezados tengan un distinto color verde claro en este caso.
