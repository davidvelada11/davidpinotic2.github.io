Propiedades del texto

Font family

Esta propiedad se utiliza para cambiar el tipo de fuente mediante la etiqueta font-family.Hay dos tipos de font family:

Generic font family

Son aquellos tipos de fuente que tienen todos los navegadores, aunque pueden verse distintas si se ven desde diferentes ordenadores.
Algunos ejemplos de generic font family son: sans-serif, serif o monospace, que se utiliza en el siguiente ejemplo:

Monospace

Specific font family

Para solucionar el problema de que con generic font family el tipo de fuente puede ser distinto, se utiliza specific font family,
ya que son iguales en cualquier ordenador, pero no están presentes en todos ellos, por lo que se suele añadir una fuente genérica
para que se active esa en ese caso. Un ejemplo es Helvetica:

Helvetica

Font size

Se utiliza para cambiar el tamaño de la fuente. Se hace a través de la etiqueta font-size y se puede escribir:

mediante un número de pixeles

mediante una relación de proporcionalidad con los encabezados a través de la unidad relativa em

Font weight

La etiqueta font-weight: bold se utiliza para poner en negrita todo el párrafo sin abusar de la etiqueta strong

Font style

La etiqueta font-style:italic se utiliza para poner en cursiva todo el párrafo sin abusar de la etiqueta em.

Line height

La etiqueta line-height se utiliza para cambiar el tamaño de interlineado.
Al igual que el tamaño de la fuente, se puede escribir
mediante pixeles o em si queremos que vaya proporcional
al tamaño de la fuente de los párrafos

Text-align

La etiqueta text-align permite situar la parte del texto seleccionada en el centro, izquierda o derecha de la línea.

Text decoration

La etiqueta text-decoration sirve para decorar el texto. Entre otras cosas permite subrayar o directamente declarar que no queremos decoración del texto añadiendo none.
